# GMG Khalsa School Website 🎓

A structured and responsive WordPress website developed for a client, designed to present school information, academic details, and facilities in a clear and organized way.

## 🔗 Live Demo

https://gmgkhalsaschool.org

---

## 📌 About The Project

This project was developed for a client to create a school website with a clean and informative layout. The main focus was to organize academic content, facilities, and announcements in a way that is easy to navigate for students, parents, and visitors.

The website includes multiple sections such as school information, academics, facilities, and updates, providing a complete overview of the institution.
